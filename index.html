Conexión a Supabase (base de datos)
¿Cómo conectar a tu base de datos real?
1. Crea una cuenta gratuita en supabase.com
2. Crea un proyecto y ejecuta el SQL de abajo
3. Reemplaza la URL y la API KEY en este archivo
4. Los registros se enviarán directamente a tu base de datos PostgreSQL
1. SQL para crear la tabla en Supabase
CREATE TABLE registros_temperatura (
  id           uuid             DEFAULT gen_random_uuid() PRIMARY KEY,
  fecha        date             NOT NULL,
  hora         time             NOT NULL,
  area         varchar(100)      NOT NULL,
  temperatura  numeric(5,1)     NOT NULL,
  estado       varchar(20)      CHECK(estado IN ('OK','ALERTA','CRITICO')),
  operario     varchar(100)      NOT NULL,
  turno        varchar(50),
  observaciones text,
  created_at   timestamptz       DEFAULT now()
);

-- Habilitar acceso público de lectura para Power BI
ALTER TABLE registros_temperatura ENABLE ROW LEVEL SECURITY;
CREATE POLICY "allow_all" ON registros_temperatura FOR ALL USING (true);

-- Query base para gráfico por área en Power BI:
SELECT fecha, area, AVG(temperatura) as temp_promedio,
       MIN(temperatura) as temp_min, MAX(temperatura) as temp_max,
       COUNT(*) as total_registros
FROM registros_temperatura
GROUP BY fecha, area ORDER BY fecha DESC;
Para Looker Studio (Google): Usa el conector PostgreSQL con las mismas credenciales. Crea un origen de datos y arrastra los campos al lienzo.

Para Power BI Desktop: Obtener datos → PostgreSQL → pega las credenciales de Supabase → elige la tabla registros_temperatura.
